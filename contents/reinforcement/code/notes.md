# TypeScript - Introdução
## Primeira Aula
### O que é o Typescript?

- Javascript - JS
  - Funções construtoras;
  - Tipagem dinâmica;
  - Prototypes;
  - ES6.
- Typescript - TS
  - Tipagem estática;
  - Interfaces;
  - Enums;
  - Decorators;
  - Generics;
---

### Install Typescript via npm.
~~~
npm install -g typescript
~~~

### Run the compiler via tsc.
~~~
npx tsc
~~~
