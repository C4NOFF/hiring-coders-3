Typescript Guide

Start NPM
~~~shell
npm init
~~~

Install Typescript
~~~shell
npm i typescript
~~~

Create file [tsconfig.json]
~~~shell
npx tsc --init
~~~

Remove Comment
~~~json
"outDir": "./dist/",
"rootDir": "./",
~~~

Start Compilation
~~~shell
npx tsc --watch
~~~

Install Types/Typescript
~~~shell
npm install @types/node
~~~

Install Query String
~~~shell
npm i query-string
~~~
