# | Guia do Curso |

## Iniciar um Projeto NodeJS

```bash
npm init -y
```

## Instalação do Framework Express

```bash
npm install express
```

## Instalação do Sucrase

```bash
npm install sucrase
```

## Instalação do Nodemon

```bash
npm install nodemon -D
```

## Executar Programa

```bash
node src/server.js
```
